# storybook design system sample

* install

```bash
$ npm i storybook-ghpage-deploy
```

* usage

```jsx
import { PrimaryButton, SecondaryButton, Title, Tag } from 'storybook-ghpage-deploy';

function Component() {
  return (
    <div className="App">
      <PrimaryButton />
      <SecondaryButton />
      <Title />
      <Tag />
    </div>
  );
}
```
